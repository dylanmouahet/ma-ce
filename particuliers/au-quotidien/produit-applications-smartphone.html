Vos questions / Nos réponses

iPhone : je n’arrive pas à m’identifier et mon champ est grisé. Que faire ?

Depuis la page d’identification, cliquez sur « Annuler » puis sur « Connexion » depuis le menu latéral. Le clavier numérique pour renseigner votre code confidentiel apparaîtra après la saisie de l'identifiant.

Puis-je accéder à mes comptes sur mon téléphone mobile ?

Oui, vous pouvez accéder à vos comptes depuis votre téléphone mobile. L’accès est compris dans l’abonnement au service « Direct Ecureuil Internet ».
En complément, vous pouvez être informé rapidement, par SMS, des mouvements de votre compte (envoi du solde, alerte sur solde, alerte sur l’encours de votre carte bancaire et alerte sur vos opérations créditrices), grâce au service « Alerte Ecureuil ».

Où trouver mes codes d’accès ?

Pour rappel :

  • L'identifiant client est inscrit sur votre contrat Banque à Distance et sur vos relevés de compte bancaire ;
  • Le code confidentiel vous a été transmis par courrier postal simple à votre domicile suite à l'ouverture de votre contrat Banque à Distance.

Que faire en cas d'oubli de mon code confidentiel ?

Si vous avez oublié votre code confidentiel ou que vous avez bloqué votre code après 3 tentatives infructueuses :

  • Vous pouvez contacter votre Centre de Relation Client ou votre conseiller en agence,
  • Vous pouvez cliquer dans la zone accès client "Code confidentiel oublié" et remplir le formulaire.

Nous vous adresserons un nouveau code confidentiel par courrier postal dans les meilleurs délais. Pour des raisons de sécurité, vous devrez alors le modifier lors de votre première utilisation.

L’accès à mes comptes depuis mon téléphone mobile est-il gratuit ?

L’accès est gratuit ; il nécessite seulement un abonnement Direct Ecureuil.
Vous payez néanmoins votre connexion à l’Internet mobile, dont les coûts varient selon les opérateurs. Ces connexions sont maintenant incluses dans la plupart des forfaits multimédia avec option Internet illimité. N’hésitez pas à contacter le service client de votre opérateur de téléphonie mobile afin d’en savoir plus sur l’offre dont vous disposez.

Comment savoir si mon téléphone est compatible ?

Le service est compatible avec les téléphones équipés d’un navigateur Internet mobile. En revanche, il n’est plus accessible avec les anciens navigateurs compatibles exclusivement avec les sites WAP ou i-mode.
Pour vérifier si votre téléphone est compatible, le plus simple est d’appeler le service client de votre opérateur mobile.

Est-il possible de consulter mes comptes depuis l’étranger ?

  • Pour les applications mobiles : l’accès aux comptes et à la recherche d’agences nécessitent une connexion Internet. Cependant les numéros de téléphone d’opposition et d’assistance sont accessibles sans connexion Internet.
  • Pour le site mobile : l’accès nécessite une connexion Internet.

Comment installer les applications Caisse d’Epargne sur mon téléphone mobile ?

Il suffit de vous rendre sur le store de votre smartphone : App Store pour iPhone et iPad, Android Market et de télécharger gratuitement l’application « Caisse d’Epargne ».
Si vous rencontrez des soucis en passant par le "store" de votre smartphone, connectez vous directement sur notre site mobile www.m.caisse-epargne.fr. Votre mobile sera reconnu automatiquement et vous serez orienté vers la solution mobile Caisse d’Epargne la plus adaptée.

Je n’arrive pas à me connecter à l’application. Que faire ?

Vérifiez la qualité de votre connexion Internet et l’exactitude de vos identifiants et mot de passe.
Si vous rencontrez encore des difficultés après ces vérifications, vous pouvez nous signaler ce dysfonctionnement survenu lors de l'utilisation de notre application, ou nous remonter des suggestions d’amélioration, en contactant notre support technique par e-mail : mobile@caisse-epargne.fr.

En cas de dysfonctionnement, merci de nous préciser les informations suivantes : votre identifiant client, le modèle de votre Smartphone ou Tablette et la version d'OS utilisée, la version de l'application Caisse d'Epargne utilisée (dans Infos Éditeur), ainsi que la description du dysfonctionnement rencontré. Nous vous répondrons dans les meilleurs délais.

Ce service est-il sécurisé ?

Tous les flux entre votre mobile et la Caisse d’Epargne sont sécurisés et passent par l’intermédiaire du protocole HTTPS.

  • Aucune donnée n'est stockée sur votre mobile à l’exception de votre numéro d’abonné, si vous activez cette option de mémorisation.

Grâce au mot de passe, vous êtes le seul à pouvoir consulter vos comptes. Pour plus de sécurité, vous êtes automatiquement déconnecté après quelques minutes d’inactivité.

Que faire si je vends mon téléphone mobile ?

Sur iPhone® ou iPad® : Supprimez l’application de votre téléphone en appuyant longuement sur l’icône jusqu’à ce que toutes vos applications bougent. Une croix va apparaître sur l’icône, appuyez sur cette croix pour confirmer la suppression de l’application.

  • Sur un téléphone Android : Supprimez l'application dans Paramètres > Applications > Gérer les applications ou depuis l’Android market.
  • Sur Smartphone : Supprimez les cookies de votre navigateur web.

Que faire en cas de perte ou de vol de votre téléphone mobile ?

Signalez la perte ou le vol de votre téléphone mobile à votre opérateur.

  • Le service mobile est protégé grâce à votre mot de passe Direct Ecureuil. Ce mot de passe n’est pas mémorisé par l’application. Pour rappel votre mot de passe est personnel et incessible, ne l’inscrivez sur aucun document ou support.

Puis-je changer mon mot de passe Direct Ecureuil depuis mon téléphone mobile ?

Non, pour changer votre mot de passe, pour des raisons de sécurité, vous devez vous connecter à vos comptes en ligne (Direct Ecureuil) sur le site Internet www.caisse-epargne.fr.
